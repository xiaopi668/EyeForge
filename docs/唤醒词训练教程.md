# 自定义唤醒词训练教程

EyeForge 使用 Picovoice Porcupine 实现离线语音唤醒。使用前需要先**获取 Picovoice AccessKey（密钥）**，然后可以配合内置唤醒词或自定义唤醒词使用。

## 获取 AccessKey

所有功能（包括内置唤醒词）都需要 Picovoice AccessKey：

1. 打开 [Picovoice Console](https://console.picovoice.ai/)
2. 点击 **Sign Up** 注册（支持 Google / GitHub 登录）
3. 登录后在控制台首页即可看到 **AccessKey**（以 `你的邮箱/` 开头的一串字符）
4. 复制该密钥，粘贴到 EyeForge 设置 → 常规 → **Picovoice AccessKey / 密钥** 中

> 免费 AccessKey 永久有效，无需付费。

## 内置唤醒词

以下唤醒词无需训练，直接可用：

`computer`, `hey google`, `alexa`, `hey siri`, `porcupine`, `grasshopper`, `terminator`, `bumblebee`, `picovoice`, `blueberry`

在设置 → 常规 → 唤醒词中填入即可。

## 训练自定义唤醒词

### 第一步：注册 Picovoice 账号

同上 [Picovoice Console](https://console.picovoice.ai/)，登录后获取 AccessKey。

### 第二步：创建自定义唤醒词

1. 在控制台左侧选择 **Wake Word** → **Create**
2. 输入你想要的唤醒词（如 `Hey Forge`、`小爱同学`、`你好助手`）
3. 选择语言（支持中文、英文等多语言）
4. 点击 **Create**，系统会生成该唤醒词的音频模型

> 免费账号每月只能创建 **1 个**自定义唤醒词，次月重置。

### 第三步：下载 .ppn 文件

1. 创建成功后，在唤醒词列表中点击 **Download**
2. 选择 **Porcupine** 平台
3. 选择 **Python** 版本
4. 下载生成的 `.ppn` 文件

### 第四步：放入项目

1. 将下载的 `.ppn` 文件放到 `EyeForge/` 目录下（或任意位置）
2. 在设置 → 常规 → 唤醒词中填入 `.ppn` 文件的**绝对路径**

示例唤醒词配置：

```
C:\Users\YourName\EyeForge\hey-forge.ppn
```

> 自定义唤醒词需要使用 `.ppn` 文件路径，内置唤醒词直接输入名称即可。多个唤醒词用逗号分隔。

### 第五步：测试

1. 确保已在 Picovoice AccessKey 字段填入密钥
2. 开启语音唤醒
3. 说出你训练的唤醒词
4. 悬浮窗应自动弹出并开始录音

## 注意事项

- **AccessKey 是必需项**，包括使用内置唤醒词也需要
- Picovoice 免费账号每月可创建 **1 个**自定义唤醒词（内置唤醒词不受限制）
- 自定义唤醒词文件（`.ppn`）是平台绑定的，下载时请选择 Python 版本
- 唤醒词训练一次后永久可用，无需联网
- 如需更多唤醒词，可以考虑升级 Picovoice 付费计划
